Info 1) 
Assessment 2 focuses on implementing two core programs: a text encryption and decryption system, and a mathematical expression evaluator.

Info 2) 
The encryption program reads an input text file, applies a custom character shifting algorithm, and generates an encrypted version of the content. It then decrypts the encrypted text to verify correctness.

Info 3)
The expression evaluator processes arithmetic expressions using recursive descent parsing. It supports addition, subtraction, multiplication, division, parentheses, unary operators, and implicit multiplication. The evaluator also handles invalid expressions gracefully and produces structured output including tokens, parse tree, and final result.

Info 5)
This assignment tests understanding of file handling, parsing techniques, error handling, and algorithm design in Python.
